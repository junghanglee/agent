# AI Linker Platform

AI Linker는 AI Agent 설치프로그램/설치코드 판매, 저가형 LLM 토큰 충전, Skill/컴포넌트 마켓, 구매자 커뮤니티, 관리자 백오피스를 제공하는 통합 플랫폼입니다.

## 현재 우선순위

1. 반응형 사용자 웹서비스
2. 슈퍼관리자 대시보드/백오피스
3. 상품/릴리즈/설치코드/결제/토큰/상담/커뮤니티 관리
4. 추후 Agent별 Windows/iOS 설치형 실행파일 등록

## 핵심 판매 상품

- AI Agent 쉬운 설치 지원 설치프로그램
- 설치코드/라이선스
- 시장가격보다 50% 이상 저렴한 LLM 토큰 충전 시스템
- AI Agent 튜닝/업그레이드용 Skill 및 컴포넌트
- 구매자 커뮤니티/상담/채팅 지원
