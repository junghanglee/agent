# 01. AI Linker 비즈니스 모델

## 1. 서비스 정의

AI Linker는 초보자도 AI Agent를 쉽게 설치하고 사용할 수 있도록 돕는 플랫폼입니다. 사용자는 웹에서 목적에 맞는 AI Agent 설치프로그램을 고르고, 설치코드와 함께 구매한 뒤, 자신의 Windows 또는 iOS 환경에 설치합니다. 설치 후에는 AI Linker의 LLM 토큰 충전 시스템, Skill/컴포넌트 마켓, 커뮤니티, 상담 기능을 통해 지속적으로 Agent를 업그레이드하고 운영합니다.

## 2. 핵심 판매 상품

### 2.1 AI Agent 설치프로그램

- Agent별 완성형 설치 실행파일 제공
- Windows 버전과 iOS 버전을 각각 별도 제작
- 예: Hermes Windows 설치파일, Hermes iOS 실행형 앱/컨트롤러
- 최종 릴리즈는 슈퍼관리자가 등록
- 구매자는 최신 릴리즈 정보와 함께 최신 설치파일 다운로드

### 2.2 설치코드 / 라이선스

- 구매 후 설치코드 발급
- 설치프로그램에서 설치코드 입력 또는 자동입력
- 설치 가능 횟수, 기기 제한, 라이선스 상태 관리
- 결제/승인 상태에 따라 실행 가능 여부 제어

### 2.3 LLM 토큰 충전 시스템

- 시장가격보다 50% 이상 저렴한 LLM 사용권 제공을 목표
- 사용자는 직접 복잡한 LLM 계정을 만들지 않고 AI Linker 토큰/크레딧을 충전
- AI Linker는 보유한 LLM 계정 Pool 또는 Gateway를 통해 요청 처리
- 토큰제, 크레딧제, 정액제 모두 지원 가능

### 2.4 Skill / 컴포넌트 마켓

- Agent 기능 확장용 Skill과 컴포넌트 판매
- 예: 메신저 연결, 일정관리, 문서작성, 고객상담, 웹검색, 업무자동화
- 무료/유료/구독형/패키지형 제공 가능

### 2.5 커뮤니티 / 상담 / 채팅

- 구매자간 소통
- Agent별 Q&A
- 설치 문제 상담
- 튜닝 요청
- 운영 노하우 공유
- 관리자 또는 상담원 채팅 지원

## 3. 서비스 수익 모델

1. 설치프로그램 판매
2. 설치코드/라이선스 판매
3. LLM 토큰 충전 수수료/마진
4. 정액제 LLM 구독
5. Skill/컴포넌트 판매
6. Agent 튜닝/업그레이드 서비스
7. 기업용 관리형 구독
8. 프리미엄 상담/세팅 대행

## 4. 현재 개발 우선순위

현재는 개별 설치파일을 계속 만드는 것보다, 설치파일을 상품으로 등록하고 판매/관리할 수 있는 웹서비스와 슈퍼관리자 백오피스를 먼저 구축합니다.

우선 개발 대상:

- 반응형 사용자 웹서비스
- 슈퍼관리자 대시보드
- 상품/릴리즈 관리
- 설치코드/라이선스 관리
- 결제/충전 관리
- LLM 계정 Pool 관리
- 사용량 모니터링
- 상담/채팅/커뮤니티 관리
