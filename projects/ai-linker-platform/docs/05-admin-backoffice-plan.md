# 05. 슈퍼관리자 백오피스 기능계획

## 1. 관리자 역할

### 1.1 슈퍼관리자

- 전체 서비스 설정
- 모든 고객/상품/결제/LLM 계정 관리
- 관리자 계정 생성/권한 부여
- 릴리즈 승인
- 정산/매출 확인

### 1.2 운영관리자

- 고객 문의 처리
- 결제/환불 처리
- 설치코드 재발급
- 상품 정보 수정
- 커뮤니티 관리

### 1.3 기술관리자

- Agent 릴리즈 등록
- 설치파일 업로드
- LLM 계정 Pool 관리
- 장애/로그 확인
- Gateway 상태 확인

### 1.4 상담관리자

- 1:1 문의
- 실시간 채팅
- 설치 문제 대응
- FAQ/공지 작성

## 2. 대시보드

표시 지표:

- 오늘 매출
- 월 누적 매출
- 신규 가입자
- 신규 구매자
- 설치코드 발급 수
- 활성 라이선스 수
- 토큰 충전액
- LLM 원가
- 예상 마진
- 미처리 상담
- 장애 알림

## 3. 고객관리

기능:

- 고객 목록 검색/필터
- 고객 상세
- 가입일/최근접속
- 보유 Agent
- 설치코드
- 라이선스 상태
- 토큰 잔액
- 사용량
- 결제내역
- 상담내역
- 관리자 메모

관리 액션:

- 라이선스 활성/정지
- 설치코드 재발급
- 크레딧 수동 지급/차감
- 고객 상태 변경
- 메모 작성

## 4. 상품관리

Agent 상품 관리:

- 상품명
- 설명
- 카테고리
- 목적 태그
- 숙련도
- 지원 OS
- 가격
- 공개/비공개
- 대표 이미지
- 상세 이미지
- 영상 URL
- 포함 Skill
- 추가 Skill

릴리즈 관리:

- Agent 선택
- OS 선택: Windows/iOS
- 버전
- 릴리즈 노트
- 설치파일 업로드
- 파일 해시
- 파일 크기
- 최신 버전 지정
- 강제 업데이트 여부

## 5. 결제관리

기능:

- 결제내역
- 결제 상태
- 환불
- 부분환불
- 충전내역
- 구독 상태
- 정액제 플랜
- 세금계산서/영수증 정보

결제 상태:

```text
pending
paid
failed
cancelled
refunded
partially_refunded
```

## 6. 설치코드/라이선스 관리

설치코드 상태:

```text
issued
used
expired
revoked
```

라이선스 상태:

```text
pending_payment
active
suspended
expired
revoked
```

기능:

- 설치코드 자동 생성
- 수동 발급
- 기기 등록 확인
- 설치 횟수 제한
- 라이선스 정지/재개
- 관리자 승인

## 7. LLM 계정 Pool 관리

Provider 관리:

- OpenAI
- Anthropic
- OpenRouter
- Gemini
- Groq
- 기타

계정 관리:

- Provider
- 계정명
- API Key 별칭
- 암호화 저장 여부
- 사용 가능 모델
- 월 한도
- 현재 사용량
- 상태
- 우선순위

라우팅 정책:

- 비용 최저 우선
- 품질 우선
- 장애 우회
- 특정 상품 전용 계정
- 정액제 전용 계정

## 8. 사용량 모니터링

집계 기준:

- 사용자별
- Agent별
- Skill별
- 모델별
- Provider별
- 일/월 단위

지표:

- 입력 토큰
- 출력 토큰
- 총 토큰
- 원가
- 판매가

- 실패 요청
- 평균 응답시간

## 9. 상담/채팅 관리

기능:

- 문의 티켓
- 실시간 채팅
- 상담 상태
- 담당자 배정
- 답변 템플릿
- 설치문제 카테고리
- 고객 구매정보 사이드패널

상담 상태:

```text

pending_user
pending_admin
resolved
closed
```

- Agent별 게시판
- 차단/숨김
