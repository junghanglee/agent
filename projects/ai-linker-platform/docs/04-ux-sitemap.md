## 3. 사용자 핵심 UX 흐름

### 3.1 초보자 추천 흐름

```text
랜딩 → “무엇을 하고 싶나요?” 선택 → 숙련도 선택 → 추천 Agent 카드 → 상세페이지 → 구매 → 설치코드/다운로드
```

UX 포인트:

- 기술 용어 최소화
- “내 목적에 맞는 Agent 찾기”를 전면 배치
- OS별 설치 가능 여부를 명확히 표시
- 구매 전 필요한 계정/비용/제한사항 안내

### 3.2 구매 후 설치 흐름

```text
결제완료 → 설치코드 발급 → 최신 릴리즈 다운로드 → 설치 매뉴얼 확인 → 설치프로그램 실행 → 설치 완료 → Console 이동
```

UX 포인트:

- 결제완료 화면에서 바로 설치코드 복사 가능
- Windows/iOS 다운로드 버튼 분리
- 최신 버전과 릴리즈노트 표시
- 설치 실패 시 고객지원으로 연결

### 3.3 토큰 충전 흐름

```text
마이페이지 → 잔액 확인 → 충전 상품 선택 → 결제 → 크레딧 지급 → 사용량 확인
```

UX 포인트:

- 시장가 대비 절약률 표시
- 예상 사용 가능량 표시
- 잔액 부족 전에 알림
- 정액제/충전형 비교 카드 제공

## 4. 디자인 방향

키워드:

```text
현대적
미니멀
고급스러움
초보자 친화적
명확한 정보구조
쇼핑몰 같은 구매 편의성
SaaS 대시보드 같은 신뢰감
```

권장 스타일:

- 배경: 아주 연한 그레이/블루 계열
- 주요 색상: 딥 네이비 + 바이올렛 + 시안 포인트
- 카드형 레이아웃
- 넓은 여백
- 단계형 진행 UI
- 가격/상태/OS 배지는 명확하게
- 관리자 화면은 표와 필터 중심

5. 마이페이지
6. 토큰 충전 페이지
7. 관리자 대시보드
8. 상품/릴리즈 관리
9. 고객/결제 관리
