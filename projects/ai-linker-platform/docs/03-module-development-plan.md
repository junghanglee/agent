# 03. 모듈 단위 개발계획

## 개발 원칙

- 한 번에 전체를 만들지 않고, 승인 가능한 작은 모듈 단위로 제작합니다.
- 각 모듈은 화면, 데이터, API, 관리자 기능, 테스트 기준을 포함합니다.
- 사장님이 각 단계별로 확인/승인/피드백하면 다음 단계로 이동합니다.

---

# Module 0. 프로젝트 기반 구축

## 목표

AI Linker 웹서비스와 백오피스를 만들 기본 프로젝트를 생성합니다.

## 산출물

- Next.js 기반 반응형 웹 프로젝트
- 관리자/사용자 라우팅 구조
- 공통 디자인 시스템
- 샘플 데이터
- 로컬 실행 가능 상태

## 승인 기준

- 메인 페이지 접속 가능
- 관리자 페이지 접속 가능
- 모바일/PC 반응형 기본 레이아웃 확인 가능

---

# Module 1. 디자인 시스템 / UX 프레임

## 목표

현대적이고 미니멀하지만 가독성 좋은 AI Linker 디자인을 확정합니다.

## 기능

- 컬러 팔레트
- 버튼 스타일
- 카드 UI
- 상품 리스트 UI
- 관리자 테이블 UI
- 상태 배지
- 가격 카드
- 반응형 네비게이션

## 승인 기준

- 랜딩/상품목록/관리자 대시보드 기본 목업 확인
- 전체 분위기 승인

---

# Module 2. 사용자 랜딩 / Agent 탐색

## 목표

사용자가 목적과 숙련도에 따라 적합한 Agent 설치프로그램을 찾을 수 있게 합니다.

## 기능

- 랜딩 페이지
- 목적별 추천
- 숙련도별 추천
- Agent 목록
- 필터/검색
- 인기/최신 릴리즈 표시

## 승인 기준

- 사용자가 쇼핑몰처럼 Agent를 찾는 흐름 확인

---

# Module 3. Agent 상세 / 릴리즈 다운로드

## 목표

상품 상세 설명과 최신 설치파일 다운로드 구조를 구현합니다.

## 기능

- Agent 상세페이지
- Windows/iOS 버전 선택
- 최신 릴리즈 정보
- 릴리즈 노트
- 설치파일 다운로드 버튼
- 설치코드 안내
- 호환 Skill 표시

## 승인 기준

- Hermes 상품 상세페이지 예시 완성
- 최신 릴리즈 다운로드 UI 확인

---

# Module 4. 회원 / 마이페이지

## 목표

구매자 계정과 구매내역을 관리합니다.

## 기능

- 회원가입/로그인
- 마이페이지
- 구매한 Agent 목록
- 설치코드 확인
- 라이선스 상태
- 다운로드 내역
- 토큰 잔액
- 사용량 요약

## 승인 기준

- 샘플 사용자로 로그인 후 구매내역 확인 가능

---

# Module 5. 결제 / 설치코드 / 라이선스

## 목표

상품 결제 후 설치코드와 라이선스를 발급합니다.

## 기능

- 결제 페이지 UI
- 결제 상태 관리
- 설치코드 생성
- 라이선스 활성/정지/만료
- 기기 제한 정책
- 관리자 승인 옵션

## 승인 기준

- 테스트 결제 또는 결제 목업으로 설치코드 발급 확인

---

# Module 6. 토큰 충전 / LLM 사용량

## 목표

저렴한 LLM 토큰 충전 시스템의 사용자 화면과 관리자 정책을 만듭니다.

## 기능

- 토큰/크레딧 잔액
- 충전 상품
- 정액제 상품
- 사용량 그래프
- 충전 내역
- 잔액 부족 안내
- 요금 정책 관리

## 승인 기준

- 사용자가 토큰 충전 상품을 보고 구매 흐름을 이해 가능

---

# Module 7. 슈퍼관리자 대시보드

## 목표

서비스 운영자가 전체 고객/상품/결제/상담/LLM 사용량을 관리합니다.

## 기능

- 매출 요약
- 신규 가입자
- 구매자 수
- Agent별 판매량
- 토큰 사용량
- LLM 비용/마진
- 장애/문의 현황

## 승인 기준

- 운영자가 하루 상태를 한 화면에서 볼 수 있음

---

# Module 8. 상품/릴리즈 관리자

## 목표

관리자가 Agent 상품과 설치파일 릴리즈를 등록합니다.

## 기능

- Agent 상품 CRUD
- Windows/iOS 릴리즈 등록
- 버전 관리
- 릴리즈 노트
- 파일 업로드
- 최신 버전 지정
- 공개/비공개

## 승인 기준

- Hermes 최신 설치파일을 관리자 화면에서 등록 가능

---

# Module 9. 고객/결제/상담 관리자

## 목표

운영 업무를 처리합니다.

## 기능

- 고객 목록
- 고객 상세
- 구매내역
- 결제내역
- 환불/정지
- 상담 티켓
- 채팅 기록
- 관리자 메모

## 승인 기준

- 특정 고객의 구매/상담/라이선스 상태를 한 화면에서 확인 가능

---

# Module 10. AI 계정 Pool / LLM Gateway 관리자

## 목표

AI Linker가 보유한 LLM 계정과 사용량을 관리합니다.

## 기능

- LLM Provider 등록
- API Key 저장 정책
- 계정 Pool 상태
- 모델별 원가/판매가
- 사용자별 사용량
- 계정별 사용량
- 라우팅 정책
- 장애 계정 제외

## 승인 기준

- 관리자 화면에서 LLM 계정 Pool과 사용량을 모니터링 가능

---

# Module 11. Skill/컴포넌트 마켓

## 목표

Agent 튜닝/업그레이드 상품을 판매합니다.

## 기능

- Skill 목록
- Skill 상세
- 호환 Agent
- 구매
- 다운로드/설치 안내
- 구매자 Q&A

## 승인 기준

- 유료 Skill 상품 구매 흐름 확인

---

# Module 12. 커뮤니티 / 채팅

## 목표

구매자간 소통과 고객지원을 제공합니다.

## 기능

- Agent별 게시판
- 공지사항
- Q&A
- 댓글
- 1:1 상담
- 관리자 답변
- 채팅 상담

## 승인 기준

- 사용자가 문의를 남기고 관리자가 답변 가능

---

# Module 13. 설치파일 템플릿 연동

## 목표

향후 Agent별 Windows/iOS 설치파일을 상품 릴리즈로 연결합니다.

## 기능

- 설치파일 메타데이터
- OS별 다운로드
- 버전별 다운로드
- 설치코드 API 연결
- 설치프로그램에서 서버 검증 가능 구조

## 승인 기준

- Hermes 설치파일이 AI Linker 상품 릴리즈로 연결됨

---

# Module 14. QA / 배포 / 운영 준비

## 목표

실서비스 가능한 수준으로 안정화합니다.

## 기능

- 테스트 시나리오
- 오류 페이지
- 로그/모니터링
- 관리자 권한 분리
- 보안 점검
- 백업 정책
- 배포 문서

## 승인 기준

- 테스트 환경 배포
- 주요 사용자/관리자 흐름 정상 작동
