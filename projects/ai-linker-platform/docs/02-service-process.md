# 02. 서비스 이용 절차와 프로세스

## 1. 사용자 구매/설치 프로세스

```text
방문
 → 목적/숙련도 기반 Agent 탐색
 → 상품 상세 확인
 → Windows/iOS 버전 선택
 → 결제
 → 설치코드 발급
 → 최신 릴리즈 다운로드
 → 설치프로그램 실행
 → 설치코드 입력/검증
 → Agent 설치
 → AI Linker 계정 연결
 → 토큰 충전/기본 크레딧 확인
 → Skill 추가설치/튜닝
 → 커뮤니티/상담 이용
```

## 2. 사용자 웹서비스 주요 화면

### 2.1 랜딩 페이지

- AI Linker 가치 제안
- AI Agent 설치를 쉽게 해준다는 메시지
- 저렴한 LLM 토큰 충전 강조
- 목적별 추천 CTA
- 인기 Agent
- 신규 릴리즈

### 2.2 Agent 탐색 페이지

필터:

- 사용목적
- 숙련도
- 운영체제
- 가격
- 인기순
- 최신순
- 초보자 추천

### 2.3 Agent 상세 페이지

표시 항목:

- Agent 설명
- 가능한 작업
- 설치 난이도
- 지원 OS
- 최신 버전
- 릴리즈 노트
- 포함 Skill
- 추가 구매 가능 Skill
- 사용 가능한 LLM 토큰 요금
- 스크린샷/영상
- 구매 버튼
- 문의/커뮤니티 링크

### 2.4 결제 페이지

- 상품명
- OS 버전
- 라이선스 조건
- 설치코드 발급 안내
- 결제수단
- 약관 동의

### 2.5 마이페이지

- 구매한 Agent
- 설치코드
- 다운로드
- 라이선스 상태
- 토큰 잔액
- 사용량
- Skill 구매내역
- 상담내역

### 2.6 토큰 충전 페이지

- 잔액
- 사용량
- 충전 패키지
- 정액제 옵션
- 결제내역
- 예상 사용 가능량

### 2.7 Skill/컴포넌트 마켓

- Skill 목록
- 상세 설명
- 호환 Agent
- 구매/설치
- 리뷰/문의

### 2.8 커뮤니티/상담

- Agent별 게시판
- 구매자 Q&A
- 설치 문제 해결
- 관리자 공지
- 1:1 상담/채팅

## 3. 슈퍼관리자 프로세스

```text
관리자 로그인
 → 대시보드 확인
 → 고객/상품/결제/상담/사용량 관리
 → Agent 설치파일 최종 릴리즈 등록
 → 설치코드/라이선스 발급/정지
 → LLM 계정 Pool 등록
 → 토큰 가격/정책 관리
 → 사용량 모니터링
 → 장애/문의 대응
```

## 4. 설치파일 릴리즈 프로세스

```text
개발자가 Agent별 설치파일 제작
 → 내부 테스트
 → 버전/릴리즈노트 작성
 → 슈퍼관리자 업로드
 → 활성 릴리즈 지정
 → 사용자 상세페이지에 최신 버전 노출
 → 구매자 다운로드 가능
```

## 5. LLM 토큰 사용 프로세스

```text
사용자 Agent 요청
 → AI Linker Gateway 인증
 → 라이선스/잔액 확인
 → 적합한 LLM 계정 Pool 선택
 → LLM API 호출
 → 토큰 사용량 계산
 → 잔액 차감
 → 사용량 통계 반영
 → 잔액 부족 시 충전 유도
```
