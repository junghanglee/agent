# 보안 규칙

## 절대 GitHub에 올리면 안 되는 것

- 비밀번호
- API 키
- 토큰
- 개인 인증 파일
- 결제 정보
- 주민등록번호 같은 개인정보

## API 키가 필요할 때

1. `.env.example`에는 예시 이름만 적습니다.
2. 실제 키는 `.env` 파일에 적습니다.
3. `.env` 파일은 `.gitignore`에 포함되어 GitHub에 올라가지 않습니다.

## 사장님이 해야 할 일

API 키를 발급받아야 하는 경우에는 내가 단계별로 안내합니다.
그때 사장님은 발급 화면에서 키를 복사해서 `.env`에 넣으면 됩니다.
채팅창에 실제 키를 그대로 붙여넣는 것은 피하는 게 좋습니다.
